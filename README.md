# todo-list_app

Vue.jsを使ったToDoリストアプリのリポジトリです。

## インストール

1. 右上の `Fork` ボタンを押してください。
2. `#{自分のアカウント名}/todo-list_app` が作成されます。
3. 任意のディレクトリにて git clone してください。

```
$ git clone https://github.com/自分のアカウント名/todo-list_app.git
```

4. ブラウザのURLバーに、以下のように入力してください。

```
/cloneしたパス/todo-list_app/views/index.html
```

## 使い方

### 登録
入力欄にて、Shift ＋ Enter で登録できます。

### 編集
編集したいTodoをクリックし、 編集モードへ移行します。
Shift ＋ Enter で編集が完了します。

### 登録中および編集中のキャンセル
Shift + Esc または 入力欄からフォーカスが外れると、入力内容が元に戻ります。

### 削除
削除ボタンをクリックします。
